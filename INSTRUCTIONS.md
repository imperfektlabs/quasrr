# Instructions for Working With the Coding Agent

## How to Start a Session
Reference the repo docs instead of re-explaining context.

Example:
“Follow AGENTS.md and PROJECT_BRIEF.md.
Write code to scaffold the backend and frontend so Docker boots cleanly.
Stop when `/health` works and the UI loads.”

## How to Continue Work
Give one concrete task per session.

Examples:
- “Add read-only SABnzbd queue integration”
- “Add streaming availability lookup (Canada only)”
- “Add natural language parsing for search input”

## How to Review
- Review diffs, not just behavior
- Enforce constraints via AGENTS.md
- Adjust direction, then continue in next session
